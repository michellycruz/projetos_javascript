# 🟨 Projetos JavaScript

> Coletânea de miniprojetos em JavaScript puro (vanilla JS) para praticar lógica, manipulação do DOM e interatividade.

Cada pasta é um projeto independente, construído com **HTML**, **CSS** e **JavaScript** — sem frameworks. O objetivo é exercitar fundamentos da linguagem na prática, um desafio de cada vez.

---

## 📁 Projetos

| # | Projeto | O que faz |
|---|---------|-----------|
| 1 | **Conversor de Unidades** | Converte valores entre diferentes unidades de medida |
| 2 | **Relógio Digital** | Exibe a hora atual em tempo real |
| 3 | **Gerador de Citações** | Mostra citações aleatórias a cada clique |
| 4 | **Galeria com Lightbox** | Galeria de imagens com visualização ampliada em lightbox |
| 5 | **Barra de Progresso** | Barra de progresso animada e dinâmica |
| 6 | **Formulário com Validação** | Formulário com validação de campos em JavaScript |
| 7 | **Paleta de Cores** | Gera e exibe paletas de cores |
| 8 | **Gerador de CPF** | Gera números de CPF válidos (para fins de teste) |
| 9 | **Calculadora de Gorjetas** | Calcula a gorjeta e divide a conta entre pessoas |
| 10 | **Jogo da Memória** | Jogo de memória com cartas viradas |
| 12 | **Conversor JSON para CSV** | Converte dados em JSON para o formato CSV |
| 21 | **Gerador de Senhas** | Cria senhas aleatórias e seguras |

## ▶️ Como executar

```bash
# Clone o repositório
git clone https://github.com/michellycruz/projetos_javascript.git

# Acesse a pasta do projeto desejado
cd projetos_javascript/2_relogio_digital

# Abra o index.html no navegador
```

Cada projeto é autossuficiente — basta abrir o respectivo `index.html`.
